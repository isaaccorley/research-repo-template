# The Remote Sensing Challenge

Traditional methods weren't built for modern satellite constellations:

- **Manual annotation** is expensive and doesn't scale
- **Multi-spectral imagery** has dozens of bands beyond RGB
- **Temporal analysis** requires processing time series of images
- Standard vision models don't understand geospatial coordinate systems

> Deep learning brings scalable, automated analysis to remote sensing.

# Foundation Models for Remote Sensing

```python
import torch
import torch.nn as nn
from torchvision.models import resnet50

class GeoEncoder(nn.Module):
    def __init__(self, in_channels: int = 13, embed_dim: int = 768):
        super().__init__()
        self.backbone = resnet50(weights=None)
        self.backbone.conv1 = nn.Conv2d(
            in_channels, 64, kernel_size=7, stride=2, padding=3, bias=False
        )
        self.backbone.fc = nn.Linear(2048, embed_dim)

    def forward(self, x: torch.Tensor) -> torch.Tensor:
        return self.backbone(x)
```

<SpeakerNotes>
- Foundation models pretrained on millions of satellite images
- Modified ResNet backbone to handle multi-spectral input (13 Sentinel-2 bands)
- Self-supervised objectives like MAE or DINO for pretraining
- Transfer to downstream tasks with minimal fine-tuning
</SpeakerNotes>

## Semantic Segmentation

The model segments land cover types from multi-spectral imagery using a U-Net architecture with a pretrained encoder.

The decoder upsamples features back to the input resolution for pixel-wise classification.

</div>
<div>

```python
class UNet(nn.Module):
    def __init__(self, encoder, num_classes: int):
        super().__init__()
        self.encoder = encoder
        self.decoder = nn.Sequential(
            nn.ConvTranspose2d(768, 256, 4, 2, 1),
            nn.BatchNorm2d(256),
            nn.ReLU(),
            nn.ConvTranspose2d(256, 64, 4, 2, 1),
            nn.BatchNorm2d(64),
            nn.ReLU(),
            nn.Conv2d(64, num_classes, 1),
        )

    def forward(self, x):
        features = self.encoder(x)
        return self.decoder(features)
```

# Training Loop

```python
import torch
from torch.optim import AdamW
from torch.utils.data import DataLoader

model = UNet(encoder=GeoEncoder(in_channels=13), num_classes=10)
optimizer = AdamW(model.parameters(), lr=1e-4, weight_decay=0.01)
criterion = nn.CrossEntropyLoss()

for epoch in range(100):
    for batch in dataloader:
        images, labels = batch["image"].cuda(), batch["label"].cuda()
        logits = model(images)
        loss = criterion(logits, labels)
        loss.backward()
        optimizer.step()
        optimizer.zero_grad()
```

<SpeakerNotes>
- AdamW with weight decay for regularization
- Cross-entropy loss for multi-class segmentation
- Typically train for 100 epochs with cosine annealing LR schedule
- Data augmentation: random flips, rotations, color jitter
</SpeakerNotes>
